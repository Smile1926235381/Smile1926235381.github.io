HTML

HTML全称“Hyper Text Markup Language（超文本标记语言）”，是网页的标准语言。HTML并不是一门编程语言，而是一门描述性的标记语言。

CSS

CSS，指的是Cascading Style Sheet（层叠样式表），是用来控制网页外观的一门技术。

JavaScript

JavaScript，就是我们通常所说的JS。这是一种嵌入到HTML页面中的编程语言，由浏览器一边解释一边执行。
